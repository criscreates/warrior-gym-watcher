# waterloo-gym-watcher
## University of Waterloo Fitness Facility Occupancy Predictor
With this project, I aim to create a web application that will incrementally fetch facility occupancy information for each of the University of Waterloo's fitness facilities.
Specifically, webpage data from [this site](https://warrior.uwaterloo.ca/FacilityOccupancy) is stored in a database. Archival of historical occupancy data enables this application to predict facility occupancy for a given day, which is intended to assist students in planning their fitness schedules and encouraging them to make use of the University's fitness facilities.
I would like to offer a special thanks to the trainers at the fitness facilities who report the live occupancy data!

> DISCLAIMER: This project is not affiliated with the University of Waterloo and is not-for-profit. I seek purely to make it easier for students to plan their fitness schedules using historical facility occupancy information that is publicly available.
